# 字符串

字符串(character strings)在APDL中可按两种方式存储：`字符参数(character
parameter)`和`字符数组(string array)`。




在定义数组时，

- CHAR, 最多8个字符；
- STRING, 最多为IMAX个字符，IMAX为8的整数倍，最大值为248。
