## 宏包 

| 宏包名称 | 描述                                           |
|----------|------------------------------------------------|
| hvfloat  | 定义浮动体的对象、标题和旋转角度。             |
| subfloat | 定义浮动体子题目。                             |
| ctable   | 排版表和图浮动体。定义表中脚注在本表宽度之内。 |
| figsize  | 自动定义图片大小。                             |
| wrapfig  | 图形被文字包围。                               |
