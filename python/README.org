** 源配置
*** Linux

修改~/.pip/pip.conf（若没有就新建一个）：

#+BEGIN_SRC 
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
#+END_SRC

*** Windows

直接在user目录中创建一个pip目录，如：C:\Users\xx\pip，新建文件pip.ini，内容如下：

#+BEGIN_SRC 
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
#+END_SRC
