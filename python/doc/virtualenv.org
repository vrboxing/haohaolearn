- 建立虚拟环境：
  + mkdir ~/.virutalevns/myevn
  + cd ~/.virtualevns/myevn
  + virtualenv myevn
  + source myevn/bin/active # 激活
  + 激活后即可在该环境中安装需要的包
- 
- 加载已存在的虚拟环境：`M-x pyvenv-workon` 
- 激活虚拟环境：`M-x pyvenv-active` 
- 关闭虚拟环境：`M-x pyvenv-deactive`
