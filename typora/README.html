块单元

段落和行断

  • 一个段落即连续的文字行。
  • 段落由一个或多个空白行分隔。
  • 回车(Return)生成一个新的段落。

返回

标题(Header)

标题使用1-6个#开头，对应1-6级标题。如：

xxxxxxxxxx
​
# This is an H1 
​
## This is an H2 
​
###### This is an H6
​

  • 快捷键：Ctrl + 1表示1级标题，以此类推。

返回

块引用

Markdown用邮件样式>符号表示块引用。

源码：

xxxxxxxxxx
> This is a blockquote with two paragraphs. This is first paragraph.
>
> This is second pragraph. Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

渲染：

This is a blockquote with two paragraphs. This is first paragraph.

This is second pragraph. Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

  • 快捷键：Ctrl + Shif + Q表示1级标题，以此类推。

返回

列表

  • 键入*建立一个无顺序列表。（*可被+或-替代）

  • 键入1.生成一个有顺序列表。

    xxxxxxxxxx
    ## 无序列表
    *   Red
    *   Green
    *   Blue
    ​
    ## 有序列表
    1.  Red
    2.  Green
    3.  Blue

返回

任务列表

任务列表用[ ]或[x]标识，如：

xxxxxxxxxx
- [ ] a task list item
- [ ] list syntax required
- [ ] normal **formatting**, @mentions, #1234 refs
- [ ] incomplete
- [x] completed

  • a task list item

  • list syntax required

  • normal *formatting*, @mentions, #1234 refs

  • incomplete

  • completed

代码块

Typora只支持Github风格的代码块Markdown语法，而不是Markdown原始的代码块样式。

使用代码块很容易，键入```，然后回车。在```后面跟上一个语言名称，可以对代码进行相应的语法高亮渲染。

源码：

xxxxxxxxxx
```
function test() {
  console.log("notice the blank line before this function?");
}
```
​
语法高亮:
```ruby
require 'redcarpet'
markdown = Redcarpet.new("Hello World!")
puts markdown.to_html
```

渲染：

xxxxxxxxxx
function test() {
  console.log("notice the blank line before this function?");
}
xxxxxxxxxx
require 'redcarpet'
markdown = Redcarpet.new("Hello World!")
puts markdown.to_html

链接

Markdown支持两类链接：inline和reference。两种链接中，链接文字都被方括号包围。

 

行内链接

xxxxxxxxxx
This is [an example](http://example.com/ "Title") inline link.
[This link](http://example.net/) has no title attribute.

URLs

Typora支持用<>包含的地址作为链接，如<i@typora.io>变为： i@typora.io。

同时Typora亦可自动将标准地址格式变为链接。
