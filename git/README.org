* Git

Git是由Linus Torvalds在2005年创造的软件版本管理和控制软件，如今已被广
泛应用于所有文档版本控制场景中，成为团队协同工作不可或缺的工具。本项目
为Git笔记，用于记录学习资源和心得。

使用Git很容易，但学习它绝非是10分钟之内就能搞定的事。只要遵循好的教程
耐心阅读、勤于实践，相信很快就会掌握Git，给你的工作带来便利。
