# Emacs

- `C-h k`：快捷键的帮助信息
- `C-h v`: 变量的帮助信息
- `C-h f`：函数的帮助信息

```
cd ~; git clone https://github.com/redguardtoo/emacs.d.git .emacs.d
mv custom.el .custom.el
```
