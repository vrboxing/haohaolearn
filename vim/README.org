* VIM
** 翻屏 
| 键  | 功能                   | 备注 |
|-----+------------------------+------|
| C-f | 向下滚动一屏           |      |
| C-d | 向下滚动半屏           |      |
| C-b | 向上滚动一屏           |      |
| C-u | 向上滚动半屏           |      |
| zz  | 将当前行移动到屏幕中央 |      |
| zt  | 将当前行移到到屏幕顶端 |      |
| zb  | 将当前行移动到屏幕底端 |      |

** 标记
| 键     | 功能                                           | 备注 |
|--------+------------------------------------------------+------|
| m{a-z} | 标记光标所在位置，局部标记，只用于当前文件     |      |
| m{A-Z} | 标记光标所在位置，全局标记。                   |      |
|        | 标记之后，退出vim，重新启动，标记仍然有效。    |      |
| `{a-z} | 移动到标记位置                                 |      |
| '(a-z) | 移动到标记行的行首                             |      |
| ``     | 移动到上次编辑的位置                           |      |
| C-o    | 跳转到更老的位置。经试验，似乎是可以跨文档跳跃 |      |
| `"     | 移动到上次离开的地方                           |      |
| `.     | 移动到上次编辑的地方                           |      |
| :marks | 显示所有标记                                   |      |

** 选择
| 键  | 功能                   | 备注 |
|-----+------------------------+------|
| C-f | 向下滚动一屏           |      |
| C-d | 向下滚动半屏           |      |
| C-b | 向上滚动一屏           |      |
| C-u | 向上滚动半屏           |      |
